# 教育行为深度可视分析平台 - 版本更新说明 (Version 2.0)

本项目已完成深度视觉重构与功能增强，旨在打造一个具有“Premium”质感与强交互性的“科技感”教育大数据分析看板。

## 🆕 新增高级可视化组件
1.  **班级综合效能雷达图 (`ClassEfficiencyRadar.vue`)**
    *   **呈现方式**：多维雷达图。
    *   **核心功能**：对比 14 个班级在学生规模、得分、正确率、方法多样性及用时五大维度的平衡性。
    *   **技术亮点**：支持滚动图例，适配全量班级对比。

2.  **方法效能 3D 气泡图 (`MethodCompareBubble.vue`)**
    *   **呈现方式**：3D 径向渐变散点图。
    *   **核心功能**：关联分析解题方法的使用次数(X轴)、正确率(Y轴)与效率指数(气泡大小)。
    *   **技术亮点**：智能坐标轴缩放（Scale），聚焦高密集数据区，解决气泡重叠问题。

3.  **题目维度透视平行坐标图 (`QuestionParallelChart.vue`)**
    *   **呈现方式**：多维平行坐标（Parallel Coordinates）。
    *   **核心功能**：揭示题目在用时、正确率与答题总量之间的复杂关联轨迹。
    *   **技术亮点**：支持轴向刷选过滤，动态颜色映射（VisualMap）。

4.  **方法使用分布锥形图 (`MethodConicalChart.vue`)**
    *   **呈现方式**：DataV 锥形柱状图。
    *   **核心功能**：直观展示各类解题思想的流行程度。

## ✨ 交互与视觉深度增强
*   **旭日图体系升级**：重构了“答题状态分布”与“群体分布”为 ECharts 旭日图，增加了中心文字提示、径向标签旋转、动画过渡及双击复位功能。
*   **雷达图精度调优**：
    *   `MethodRadar`：引入“微距观察”模式，通过精准配置 `min/max` 量程，极大地放大了学习方法间的细微性能差异。
    *   `WeekdayRadarChart`：新增交互式 Tooltip 分条目显示，格式化数字展示。
*   **全局自适应优化**：所有图表组件均深度集成了 `ResizeListener`，确保在不同分辨率大屏下均不失真。

## 🏗️ UI 布局与 UX 交互重构
1.  **核心指标 2x2 深度看板**：
    *   将“核心指标总览”从单行升级为四项指标协同展示。
    *   直观呈现：**总记录 (231,713)**、**学生数 (1,364)**、**题目数 (38)** 及 **整体正确率 (25.01%)**。
2.  **模块布局均衡化**：
    *   “班级分析”、“学习方法分析”及“题目分析”模块均由 `grid-3` 升级为 **`grid-2` (2x2)** 布局。
    *   通过增大单个组件的展示面积，显著提升了复杂图表的可读性。
3.  **弹窗系统体验优化**：
    *   **知识点分析弹窗**：改为左右分栏布局，左侧滚动指标面板，右侧大型交互旭日图。
    *   **数据概览弹窗**：引入玻璃磨砂（Backdrop Blur）滤镜与 2x2 名片式指标卡片，提升 Premium 指数。
    *   **定位修复**：彻底修复了 Overlay 遮罩层 z-index 冲突及垂直居中定位问题。

## 🛠️ 稳定性修复
- 修复了 `WeekdayRadarChart` 等组件中的 ESLint 变量未使用错误。
- 补全了项目中缺失的组件引用与数据源转换逻辑。
- 优化了 DataV 与 ECharts 混合使用的兼容性，统一了色彩调色盘。
